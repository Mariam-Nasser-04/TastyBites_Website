# 🍽️ TastyBites Website

TastyBites is a modern and responsive restaurant website developed using **ASP.NET MVC (C#)**. The project allows users to explore the restaurant's menu, browse food categories, and place orders through a clean and user-friendly interface.

## 🚀 Features

- Responsive and modern UI
- Browse food categories
- View menu items
- Add items to cart
- Place orders
- User-friendly navigation

## 🛠️ Built With

- ASP.NET MVC (C#)
- HTML5
- CSS3
- JavaScript
- Bootstrap
- SQL Server

## 📂 Project Structure

- Controllers
- Models
- Views
- wwwroot
- Database

## ⚙️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/Mariam-Nasser-04/TastyBites_Website.git
```

2. Open the solution in Visual Studio.

3. Restore NuGet packages.

4. Update the database connection string in `appsettings.json`.

5. Run the project.
